# UnionArResult

This gem adds two or more active record array results and produces active relation result.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'union_ar_result'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install union_ar_result
